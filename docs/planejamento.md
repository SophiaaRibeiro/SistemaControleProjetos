## 🗓️ 3. Planejamento do Projeto

## Etapa 1 — Planejamento e Modelagem

Objetivo: definir estrutura do sistema.

Atividades:

* Definir escopo do projeto
* Levantar requisitos
* Modelar banco de dados
* Definir estrutura de pastas
* Criar documentação inicial

## Etapa 2 — Desenvolvimento Inicial

Objetivo: construir base do sistema.

Atividades:

* Criar estrutura do projeto
* Desenvolver layout inicial
* Implementar cadastro de projetos
* Implementar listagem de projetos

## Etapa 3 — Gestão de Tarefas

Objetivo: permitir organização das atividades.

Atividades:

* Criar módulo de tarefas
* Associar tarefas a projetos
* Definir status das tarefas
* Implementar edição e exclusão

## Etapa 4 — Controle de Progresso

Objetivo: acompanhar evolução do projeto.

Atividades:

* Calcular progresso do projeto
* Criar visualização de tarefas
* Destacar tarefas atrasadas

## Etapa 5 — Melhorias e Ajustes

Objetivo: melhorar experiência do usuário.

Atividades:

* Implementar filtros
* Melhorar interface
* Corrigir bugs
* Revisar código
