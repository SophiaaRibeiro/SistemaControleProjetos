
# 📋 Requisitos do Sistema

## Requisitos Funcionais

RF01 — O sistema deve permitir cadastro de usuários.

RF02 — O sistema deve permitir login de usuários.

RF03 — O sistema deve permitir criar novos projetos.

RF04 — O sistema deve permitir editar projetos.

RF05 — O sistema deve permitir excluir projetos.

RF06 — O sistema deve permitir visualizar detalhes do projeto.

RF07 — O sistema deve permitir criar tarefas dentro de um projeto.

RF08 — O sistema deve permitir editar tarefas.

RF09 — O sistema deve permitir excluir tarefas.

RF10 — O sistema deve permitir alterar o status de uma tarefa.

RF11 — O sistema deve permitir definir prazo para tarefas.

RF12 — O sistema deve exibir o progresso do projeto baseado nas tarefas concluídas.

RF13 — O sistema deve permitir filtrar projetos por status.

RF14 — O sistema deve permitir pesquisar projetos pelo nome.

## Requisitos Não Funcionais

RNF01 — O sistema deve possuir interface simples e intuitiva.

RNF02 — O sistema deve apresentar boa organização do código.

RNF03 — O sistema deve funcionar em navegadores modernos.

RNF04 — O sistema deve possuir estrutura de código organizada em camadas.

RNF05 — O sistema deve permitir fácil manutenção e evolução.

RNF06 — O sistema deve possuir responsividade para diferentes tamanhos de tela.
