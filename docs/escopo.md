# 📌 1. Escopo do Projeto

## Sistema de Controle de Projetos

### Visão Geral

O Sistema de Controle de Projetos é uma aplicação web desenvolvida para auxiliar na organização, planejamento e acompanhamento de projetos e tarefas.

O sistema permitirá que usuários criem projetos, adicionem tarefas, acompanhem prazos e visualizem o progresso das atividades, facilitando a gestão de demandas e o controle do fluxo de trabalho.

### Objetivo do Sistema

Desenvolver uma plataforma simples que permita:

* Organizar projetos
* Gerenciar tarefas
* Controlar prazos
* Visualizar progresso das atividades
* Melhorar a organização de demandas

### Público-Alvo

* Estudantes
* Pequenas equipes
* Profissionais que desejam organizar tarefas e projetos

### Escopo Funcional

O sistema permitirá:

* Cadastro de usuários
* Criação e gerenciamento de projetos
* Criação e gerenciamento de tarefas
* Controle de prazos
* Visualização de progresso do projeto
* Filtros e pesquisa de projetos

### Fora do Escopo (versão inicial)

Não será implementado inicialmente:

* Permissões avançadas de usuários
* Integração com APIs externas
* Notificações por e-mail
* Aplicação mobile
